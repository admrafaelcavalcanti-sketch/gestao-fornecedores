Importar NFDs

Importe direto o relatório "DETALHAMENTO" (export de NFDs) — as colunas são reconhecidas automaticamente

Colunas esperadas: Filial NFD, Código, Cód. Fornecedor, Descrição, Fornecedor, CGC, Comprador, Data, Filial NFO, Nota de Origem, Nota de Devolução, Valor Unitário, Quantidade, Valor Total, IPI, Observação, Possui Imagem?, E-mail Enviado, Destinatários, Primeiro E-mail, Último E-mail.
Fornecedores são agrupados por nome — se a planilha trouxer vários códigos de ERP para o mesmo nome (ex: KRONA), tudo cai no mesmo cadastro. Cada NFD é uma nota: se a planilha tiver várias linhas com o mesmo número (um item por produto), o sistema soma os valores e junta tudo em um único registro. NFDs já importadas (mesmo número + fornecedor) não são duplicadas. Toda NFD nova entra como Pendente — depois você atualiza o status pelo painel do fornecedor.
